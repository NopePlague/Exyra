# About Exyra

**Exyra is a skill-exchange platform designed to turn learning into a two-way system.**

Instead of consuming content passively, users actively participate in a learning economy where:
- teaching earns credits
- learning consumes credits
- knowledge is exchanged between real people

It is not a course platform.  
It is not a video learning app.

 It is a **live human skill network.**

Built by **Aniket Kumar**

---

# Core Philosophy

Exyra is based on one core loop:

> **Learn → Teach → Earn → Learn Again → Improve**

Every user is both:
- a learner
- a teacher

There are no passive users.

---

# Credit Economy System

Exyra uses a skill-based internal currency system.

### Rules:
- Every user starts with **5 credits**
- Teaching → earns credits
- Learning → spends credits
- Teaching + learning in same session → balanced (0 change)

This ensures:
- fair exchange of value
- no free consumption system
- active participation from all users

---

# Session System

Exyra supports real-time learning through sessions.

### Types:

### 1. Request-Based Sessions
- User finds a teacher
- Sends request
- Teacher accepts
- Private 1-on-1 learning begins

### 2. Open Sessions
- User creates a learning room
- Others join
- Group learning happens

---

# Trust Score System

Each user has a dynamic trust score:
T = (Q × 0.35) + (R × 0.35) + (A × 0.15) + (C × 0.15)


### Components:

- **Q** → Skill verification (future tests)
- **R** → User ratings after sessions
- **A** → Activity level (credits + participation)
- **C** → Completion quality of sessions

 This ensures credibility and prevents fake reputation.

---

# Future Test System

Exyra will include skill testing modules:

- quizzes
- practical evaluations
- skill verification tasks

These directly contribute to **Q (Skill Score)** in trust system.

# System Flow

1. User signs up (minimal data)
2. User explores skills via discover page
3. User requests or joins session
4. Chat + learning happens
5. Session ends → rating + credits updated
6. Trust score updates automatically
